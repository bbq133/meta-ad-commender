## 关键改进

### 1. 图表更接近正方形
- **高度**：从 400px → 200px（减少 50%）
- **宽高比**：更加均衡，接近 1:1
- **固定高度**：使用 `max-h-[200px]` 确保不会过高

### 2. 极致压缩
- **整体间距**：gap-6 → gap-3（减少 50%）
- **卡片内边距**：平均减少 60%
- **字号**：平均减少 25-30%
- **图标**：平均减少 30-40%

### 3. 文字简化
- "Spend Threshold ($)" → "Spend ($)"
- "{config.targetType} Threshold" → "{config.targetType}"
- "All Campaigns" → "All"
- "Avg Spend" → "Avg"
- "Target {config.targetType}" → "Target"

### 4. 视觉优化
- 减小圆角（rounded-2xl → rounded-lg → rounded）
- 减小边框（border-2 → border）
- 减小间距（所有 gap 和 margin 减半）
- 添加 `leading-tight` 确保文字紧凑
